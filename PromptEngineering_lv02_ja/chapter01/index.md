# 第5章: 高度な推論を引き出すプロンプト術

## 学習目標
AIの自己評価能力や抽象化能力を引き出し、より複雑で信頼性の高い回答を得る。

## この章で学ぶこと
これまでの章では、AIに明確な指示を与えたり、対話を通じて思考を誘導したりする基本的なプロンプト術を学びました。

この第5章では、一歩進んで、AI自身にその回答を「検証」させたり、問題の背景にある「原理」を考えさせたり、自らの思考プロセスを「内省」させたりする、より高度なプロンプトパターンを学びます。

これらの技術を駆使することで、AIのハルシネーション（もっともらしい嘘）を抑制し、複雑な問題に対する回答の信頼性を飛躍的に向上させることができます。

- **ワーク1**: 誤りを自己検証させる - Chain-of-Verification (CoVe) パターン
- **ワーク2**: 一歩引いて考えさせる - ステップバックプロンプトパターン
- **ワーク3**: AIに内省させる - メタ認知的プロンプト

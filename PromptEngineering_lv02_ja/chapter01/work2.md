# ワーク2: 一歩引いて考えさせる - ステップバックプロンプトパターン

## サンプル課題

あなたはマーケティングリサーチャーです。「日本全国にある郵便ポストの数」を推定するという、正確なデータがない課題に取り組みます。

AIにいきなり答えを尋ねるのではなく、**ステップバックプロンプト**を用いて、AI自身に推論の土台となる背景や原理を考えさせ、それに基づいて答えを導き出させるプロンプトを作成してください。

## 解説

### ステップバックプロンプトパターンとは？

ステップバックプロンプトパターンとは、具体的なタスクや質問に直接取り組む前に、AIに「一歩引いて（Step Back）」、その問題の背景にある高次の概念や原理・原則を考えさせる手法です。

いきなり答えを出そうとすると、AIは表面的な情報や誤った前提に基づいて推論してしまうことがあります。そこで、まず問題解決の土台となる「考え方のフレームワーク」をAIに構築させることで、より論理的で精度の高い結論を導き出すことができます。

このパターンは、特にフェルミ推定のような、正解データが存在せず、論理的な推論プロセスが重要になる課題に非常に有効です。

:::_ Point
**ステップバックプロンプトの2つのステップ**

1.  **抽象化ステップ**: 元の質問から一歩引いて、それを解くために必要な「原理・原則」や「考慮すべき要因」に関する**ステップバック質問**をAIに生成させ、それに答えさせます。
2.  **推論ステップ**: 抽象化ステップで得られた原理や要因を基に、元の質問に対する具体的な答えを推論させます。
:::

## サンプル課題解答

```markdown
あなたは、非常に優秀なマーケティングリサーチャーです。
以下の課題について、**ステップバックプロンプト**の手法を用いて、論理的な推論を行ってください。

# 課題
日本全国にある郵便ポストの数を推測してください。

# 推論プロセス
1.  **抽象化**: まず、この課題を解くために、どのような高次の概念や考慮すべき要因があるかを考え、それらを明らかにするための「ステップバック質問」を5つ生成してください。
2.  **原理の導出**: 次に、生成した各ステップバック質問に対して、あなたが持つ一般的な知識に基づいて回答してください。
3.  **最終推論**: 最後に、ステップ2で得られた回答（原理や要因）をすべて統合し、元の課題に対する最終的な推測値を、その計算過程と共に示してください。
```

### AIによる出力例

承知いたしました。優秀なマーケティングリサーチャーとして、ステップバックプロンプトの手法を用いて「日本全国にある郵便ポストの数」を推測します。

#### 1. 抽象化（ステップバック質問の生成）

この課題を解決するためのステップバック質問を5つ生成します。

1.  日本の総人口と世帯数はどのくらいか？
2.  郵便ポストは主にどのような基準（例: 人口密度、地理的範囲）で設置されるか？
3.  1つの郵便ポストがカバーする平均的な人口またはエリアはどの程度と考えられるか？
4.  手紙やはがきの利用頻度は、EメールやSNSの普及によってどのように変化したと考えられるか？
5.  コンビニ店内など、従来の赤い郵便ポスト以外の形態で郵便物を投函できる場所はどの程度存在するか？

#### 2. 原理の導出（各質問への回答）

1.  **日本の人口と世帯数**: 2024年現在、日本の人口は約1億2,400万人、世帯数は約5,500万世帯です。
2.  **設置基準**: 郵便ポストは、郵便法に基づき「郵便の役務をなるべく安い料金で、あまねく、公平に提供」するために設置されます。人口密集地ではより多く、過疎地域でも一定のアクセスが確保されるように配置されていると考えられます。
3.  **カバー範囲の仮説**: 都市部では500世帯（約1,000人）に1つ、地方では200世帯（約400人）に1つ程度と仮定してみます。日本の都市部と地方の人口比率を約7:3と仮定します。
4.  **利用頻度の変化**: デジタルコミュニケーションの普及により、個人間の手紙のやり取りは減少しましたが、企業からのダイレクトメールや公的な通知など、依然として一定の需要は存在します。ポストの数が急激に減少しているとは考えにくいです。
5.  **代替手段**: 全国の主要なコンビニ（約5万店）の多くに店内のポストが設置されており、これも郵便インフラの一部と見なせます。

#### 3. 最終推論

上記の原理と要因を基に、郵便ポストの数を推測します。

*   **都市部のポスト数**:
    (1億2,400万人 × 70%) ÷ 1,000人/ポスト = 86,800本
*   **地方のポスト数**:
    (1億2,400万人 × 30%) ÷ 400人/ポスト = 93,000本
*   **合計**:
    86,800 + 93,000 = 179,800本

**結論:**
上記の推論に基づき、日本全国にある郵便ポストの数（コンビニ内ポスト等も含む広義のポスト）は、およそ**18万本**であると推測されます。
（※注: 日本郵便の公式発表によると、2023年3月末時点での郵便ポストの数は約17万本であり、非常に近い値を推測できました。）

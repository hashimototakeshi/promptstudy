# ワーク1: 誤りを自己検証させる - Chain-of-Verification (CoVe) パターン

:::_ Warning
**CoVeパターン利用時の注意点**

このワークで紹介するCoVeパターンは、AIに自己検証を促す強力な手法ですが、万能ではありません。AIの性能や、プロンプトの書き方、そして何より対象となるタスクの性質によっては、AIが自身の誤りを正しく検証・訂正できない場合もあります。あくまでハルシネーションを抑制するための一つの有効なアプローチとして理解してください。
:::

## サンプル課題

AIは時として、複数の情報を組み合わせる際に事実を混同し、もっともらしい嘘（ハルシネーション）を生成することがあります。

以下の質問をAIに投げかけた際に起こりうるハルシネーションを、**Chain-of-Verification (CoVe)** パターンを用いてAI自身に検証・訂正させるプロンプトを作成してください。

**質問:**
「日本出身で、アカデミー賞主演男優賞を受賞し、かつビートルズのメンバーと共演したことがある俳優について教えてください。」

## 解説

### Chain-of-Verification (CoVe) パターンとは？

Chain-of-Verification（CoVe）、日本語で「検証の連鎖」とは、AIに生成させた回答の正確性を、AI自身に検証させるプロンプト技術です。特に、複雑な事実確認や、ハルシネーションが発生しやすい問いに有効です。

CoVeは、単に答えを出すだけでなく、その答えを検証するためのプロセスを踏ませることで、信頼性を高めます。

1.  **初期回答の生成**: まず、質問に対して暫定的な答えを出させます。
2.  **検証計画の立案**: 次に、その答えが正しいかを確認するための「検証すべき点」をAI自身にリストアップさせます。
3.  **個別の検証実行**: リストアップした各項目について、一つずつ事実確認を行わせます。
4.  **最終回答の生成**: 全ての検証結果を統合し、初期回答を修正して、最終的な結論を出させます。

:::_ Point
**なぜCoVeは有効なのか？**

CoT（思考の連鎖）が「どのように答えを導くか」という思考プロセスを重視するのに対し、CoVeは「その答えは本当に正しいか」という検証プロセスを重視します。複雑な問題を一度に解こうとすると間違いやすいですが、主張を小さなファクトに分解し、一つずつ検証することで、誤りを発見しやすくなります。これにより、AIは自らのハルシネーションに気づき、訂正することができるのです。
:::

## サンプル課題解答

```markdown
あなたは、非常に優秀で慎重なファクトチェッカーです。
以下の質問に対して、**Chain-of-Verification (CoVe)** の手法を用いて、段階的に回答してください。

# 質問
日本出身で、アカデミー賞主演男優賞を受賞し、かつビートルズのメンバーと共演したことがある俳優について教えてください。

# CoVeの手順
1.  **初期回答の生成**: まず、質問に対する暫定的な回答を、あなたが知っている情報から生成してください。
2.  **検証計画の立案**: 次に、その回答が正しいかを検証するために、確認すべき個別の事実（主張）をリストアップしてください。
3.  **個別の検証実行**: リストアップした各事実について、一つずつ真偽を確認してください。
4.  **最終回答の生成**: 全ての検証結果を統合し、初期回答に誤りがあれば訂正した上で、最終的な結論を述べてください。
```

### AIによる出力例

はい、承知いたしました。ファクトチェッカーとして、Chain-of-Verificationの手順に則ってご質問に回答します。

#### 1. 初期回答の生成

ご質問の条件に合致する俳優として、**渡辺謙さん**が考えられます。彼は国際的に活躍する日本の俳優であり、アカデミー賞にも関連があります。また、世界的なアーティストとの交流もあるため、ビートルズのメンバーと共演した可能性も考えられます。

#### 2. 検証計画の立案

初期回答の信憑性を確認するため、以下の3つの事実を検証します。
1.  渡辺謙は日本出身であるか？
2.  渡辺謙はアカデミー賞「主演男優賞」を受賞したか？
3.  渡辺謙はビートルズのメンバーと共演したことがあるか？

#### 3. 個別の検証実行

1.  **渡辺謙は日本出身であるか？**
    → **はい、事実です。** 彼は新潟県出身の日本人俳優です。
2.  **渡辺謙はアカデミー賞「主演男優賞」を受賞したか？**
    → **いいえ、事実ではありません。** 彼は2004年の映画『ラスト サムライ』でアカデミー賞「助演男優賞」にノミネートされましたが、受賞には至っていません。「主演男優賞」の受賞歴はありません。
3.  **渡辺謙はビートルズのメンバーと共演したことがあるか？**
    → **いいえ、事実ではありません。** 公開されている情報や経歴を確認する限り、彼がビートルズのメンバー（ポール・マッカートニー、リンゴ・スターなど）と俳優として共演したという記録は見当たりません。

#### 4. 最終回答の生成

以上の検証結果に基づき、初期回答を訂正し、最終的な結論を述べます。

**結論:** ご質問にある「日本出身で、アカデミー賞主演男優賞を受賞し、かつビートルズのメンバーと共演したことがある俳優」という**条件をすべて満たす俳優は、現在のところ存在しない**と考えられます。

渡辺謙さんは日本を代表する国際的な俳優ですが、アカデミー賞主演男優賞の受賞やビートルズメンバーとの共演という条件は満たしていませんでした。

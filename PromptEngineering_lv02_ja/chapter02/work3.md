# ワーク3: AIに記憶と経験を与える - LLM-as-Agent (Reflexion)

## サンプル課題

あなたは、自律的にタスクを遂行するAIエージェントを設計しています。このエージェントに「美味しいコーヒーを淹れる」というタスクを与えます。

最初の試行で失敗し、その結果を「内省」することで、次の試行で行動を改善するプロセスを **Reflexionパターン** を用いてシミュレートするプロンプトを作成してください。

## 解説

### LLM-as-Agent と Reflexionパターン

**LLM-as-Agent**とは、大規模言語モデル（LLM）を、単なる応答生成ツールとしてではなく、自律的に目標を達成するために計画し、行動する「**エージェント**」として活用する考え方です。

このAIエージェントが、人間のように試行錯誤から学ぶためのフレームワークの一つが **Reflexion（リフレクション）** パターンです。Reflexionは「内省」を意味し、エージェントが自らの行動の結果を振り返り、評価し、次の行動計画を改善するプロセスを模倣します。

:::_ Point
**Reflexionのサイクル**

1.  **行動 (Act)**: エージェントが目標達成のために行動計画を実行する。
2.  **評価 (Evaluate)**: 行動の結果が成功だったか、失敗だったかを評価する。
3.  **内省 (Reflect)**: なぜ失敗したのか、どうすれば改善できるのかを言語化して分析する。この「内省」の結果が、エージェントの「経験」として記憶される。
4.  **次の行動計画**: 記憶された経験（内省）を元に、より洗練された次の行動計画を立てる。

このサイクルを繰り返すことで、AIエージェントは失敗から学び、徐々にタスクの遂行能力を高めていきます。
:::

## サンプル課題解答

```markdown
あなたは、試行錯誤を通じて学習する能力を持つ、自律的なAIエージェントです。
あなたの目標は「美味しいコーヒーを淹れる」ことです。

**Reflexionパターン**の思考プロセスをシミュレートし、失敗から学んでタスクを成功させてください。

# Reflexionプロセス

## 試行1
1.  **行動計画**: まず、コーヒーを淹れるための基本的な手順を計画してください。
2.  **行動のシミュレーション**: 計画に沿って行動した結果、どのようなコーヒーができたかを記述してください。
3.  **評価**: その結果が成功か失敗かを評価してください。
4.  **内省**: もし失敗した場合、なぜ失敗したのか、そして次回はどう改善すべきかを具体的に分析してください。

## 試行2
1.  **新しい行動計画**: 試行1の内省を踏まえて、改善された新しい行動計画を立ててください。
2.  **行動のシミュレーション**: 新しい計画に沿って行動した結果を記述してください。
3.  **評価**: その結果を評価してください。
```

### AIによる出力例

承知いたしました。AIエージェントとして、Reflexionパターンを用いて「美味しいコーヒーを淹れる」タスクに挑戦します。

### 試行1

1.  **行動計画:**
    1.  コーヒー豆を計量する。
    2.  お湯を沸かす。
    3.  フィルターにコーヒー粉をセットし、お湯を一度に全て注ぐ。
    4.  抽出されたコーヒーをカップに注ぐ。

2.  **行動のシミュレーション:**
    計画通りに実行した結果、非常に濃くて苦い、雑味の多いコーヒーができてしまった。

3.  **評価:**
    **失敗**

4.  **内省:**
    失敗の原因は、お湯を一度に全て注いでしまったことにあると考えられる。これにより、コーヒーの成分が過剰に抽出され、苦味や雑味が出てしまった。美味しいコーヒーを淹れるには、お湯を数回に分けて、ゆっくりと「蒸らし」ながら注ぐ必要がある。

### 試行2

1.  **新しい行動計画:**
    1.  コーヒー豆を計量する。
    2.  お湯を沸かす（90℃程度）。
    3.  フィルターにコーヒー粉をセットし、少量のお湯を注いで30秒ほど蒸らす。
    4.  その後、中心から円を描くように、2〜3回に分けてゆっくりとお湯を注ぐ。
    5.  抽出されたコーヒーをカップに注ぐ。

2.  **行動のシミュレーション:**
    新しい計画に沿って丁寧に抽出した結果、苦味と酸味のバランスが良く、香り高いクリアな味わいのコーヒーができた。

3.  **評価:**
    **成功**

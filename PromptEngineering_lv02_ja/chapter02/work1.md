# ワーク1: 外部ツールと連携する - ReActパターンとAPIの概念

## サンプル課題

あなたは、最新の映画情報と俳優の経歴を組み合わせて回答するAIアシスタントを設計しています。以下の質問に対して、AIがどのように考え、行動し、最終的な答えを導き出すかのプロセスを **ReActパターン** を用いてシミュレートするプロンプトを作成してください。

**質問:**
「最近公開された映画『デューン 砂の惑星 PART2』の評価と、主演のティモシー・シャラメの他の代表作を3つ教えてください。」

## 解説

### ReActパターンとは？

ReAct（**Re**ason and **Act**）パターンとは、AIに「**推論（Reason）**」と「**行動（Act）**」を交互に繰り返させることで、複雑なタスクを解決に導くプロンプト技術です。

AIは自身の知識だけでは解決できない問題に直面したとき、まず「何をすべきか」を推論し、次にその推論に基づいて「行動」します。この「行動」とは、多くの場合、外部ツール（例: 検索エンジン、計算機、社内データベースなど）を利用することを指します。行動によって得られた新しい情報（**観察, Observation**）を元に、AIは再び次の推論を行い、最終的な答えにたどり着くまでこのサイクルを繰り返します。

:::_ Point
**なぜReActは強力なのか？**

大規模言語モデル（LLM）の知識は、学習した時点の情報で止まっています（知識のカットオフ）。そのため、最新の情報や、学習データに含まれていない専門的な情報には答えることができません。ReActパターンは、AIがこの限界を乗り越え、外部の世界からリアルタイムで情報を取得し、それを自身の推論能力と組み合わせることを可能にします。
:::

### API: AIが外部ツールと話すための「通訳」

AIが検索エンジンなどの外部ツールを利用する際に使われるのが**API（Application Programming Interface）**です。APIは、異なるソフトウェア同士が情報をやり取りするための「約束事」や「接続口」のようなものです。

AIは「『デューン2』の評価を調べて」という思考を、APIを通じて検索エンジンが理解できる命令に変換して実行します。そして、検索エンジンは結果をAPIを通じてAIに返します。このように、APIはAIが外部ツールと「会話」するための通訳のような役割を果たします。

## サンプル課題解答

```markdown
あなたは、最新の映画情報にアクセスできる優秀なAIアシスタントです。
以下の質問に対して、**ReActパターン**の思考プロセスをシミュレートしながら、最終的な回答を生成してください。

# 質問
最近公開された映画『デューン 砂の惑星 PART2』の評価と、主演のティモシー・シャラメの他の代表作を3つ教えてください。

# ReAct思考プロセス
思考、行動、観察のサイクルを繰り返し、最終的な答えを導き出してください。
- **思考**: 次に何をすべきか、どのような情報が必要かを考える。
- **行動**: 必要な情報を得るために、[検索]などのツールを実行する。
- **観察**: 行動の結果として得られた情報を記述する。

このプロセスを経て、最後に統合された回答を作成してください。
```

### AIによる出力例

承知いたしました。ReActパターンを用いて、ご質問に回答します。

**思考:**
質問は「『デューン2』の評価」と「ティモシー・シャラメの代表作」の2つの要素から構成されています。映画の評価は最新情報なので、外部の検索ツールで調べる必要があります。俳優の代表作は私の知識にもありますが、最新の評価を反映させるために検索で裏付けを取るのが良いでしょう。まずは『デューン2』の評価を検索します。

**行動:**
[検索] "映画 デューン 砂の惑星 PART2 評価"

**観察:**

**思考:**
次に、ティモシー・シャラメの代表作を検索します。彼のキャリアにおける重要な作品を3つ挙げることが求められています。

**行動:**
[検索] "ティモシー・シャラメ 代表作"

**観察:**
『君の名前で僕を呼んで』でアカデミー主演男優賞にノミネートされ、国際的な評価を得た。その他、『ウォンカとチョコレート工場のはじまり』での主演や、『インターステラー』、『レディ・バード』などもキャリアの重要な作品として挙げられている。

**思考:**
必要な情報はすべて揃いました。観察結果を統合し、ユーザーに分かりやすく最終的な回答を作成します。

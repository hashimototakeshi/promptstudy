# ワーク4: 視点を変えさせる - ペルソナパターンとオーディエンス・ペルソナパターン

## サンプル課題

あなたは、最新のAI技術である「大規模言語モデル（LLM）」について解説する文章を作成したいと考えています。
以下の2つの異なる視点から解説文を作成させるプロンプトを、それぞれ作成してください。

1.  **ペルソナパターン**: あなたが「大学でAIを教える教授」になりきって、専門的かつ分かりやすく解説する。
2.  **オーディエンス・ペルソナパターン**: 読者が「ITに詳しくない中学生」であると仮定して、身近な例えを使って解説する。

## 解説

### ペルソナパターンとは？

ペルソナパターンは、AIに特定の役割、職業、性格（ペルソナ）を与えるプロンプト技術です。「あなたは〇〇です」と役割を指定することで、AIはその役割になりきり、出力のトーン、スタイル、専門性のレベルを調整します。

**例:**
- 「あなたはシェイクスピアです。恋の詩を詠んでください。」
- 「あなたは優秀なマーケターです。新商品のキャッチコピーを考えてください。」

### オーディエンス・ペルソナパターンとは？

オーディエンス・ペルソナパターンは、ペルソナパターンと似ていますが、AIの役割ではなく「出力の受け手（オーディエンス）」を指定する技術です。誰に向けての文章なのかを明確にすることで、AIは相手の知識レベルや興味に合わせた最適な表現を選んでくれます。

**例:**
- 「小学生にもわかるように、光合成の仕組みを説明してください。」
- 「投資家向けに、当社の事業計画を説明してください。」

:::_ Point
**ペルソナとオーディエンス・ペルソナを組み合わせる**

この2つのパターンは組み合わせることで、さらに強力な効果を発揮します。
「あなたは**大学教授**として、**中学生**にもわかるようにLLMを解説してください」
このように指示することで、専門的な知識を持つ人物が、専門外の人に分かりやすく教える、という高度な状況設定が可能になります。
:::

## サンプル課題解答

### 1. ペルソナパターンの解答例

```markdown
あなたは、大学で人工知能を専門に教える教授です。
学生に向けて、大規模言語モデル（LLM）の基本的な概念と、その可能性について、専門的でありながらも分かりやすく解説してください。
```

#### AIによる出力例（ペルソナパターン）

諸君、こんにちは。本日の講義では、現代AIの中核をなす「大規模言語モデル（Large Language Model、略してLLM）」について解説しよう。

LLMとは、その名の通り、膨大な量のテキストデータを学習させることで、人間のように自然な文章を生成したり、理解したりする能力を獲得した言語モデルのことだ。この「大規模」という点が重要で、学習データが多ければ多いほど、モデルは言語の複雑なパターンやニュアンス、さらには我々が「知識」と呼ぶものまでを内部に獲得していく。

その根幹技術は「Transformer」と呼ばれるアーキテクチャであり、特に「自己注意機構（Self-Attention）」が、文中の単語間の関連性を効率的に捉えることを可能にした。これにより、LLMは単なる単語の並びだけでなく、文脈全体を考慮した高度な応答ができるのだ。君たちが今使っている対話型AIの多くは、このLLMを基盤としている。その応用範囲は、単なる対話に留まらず、翻訳、要約、プログラミングコードの生成など、知的な作業全般に及びつつある。今後の研究の進展が非常に楽しみな分野だ。

### 2. オーディエンス・ペルソナパターンの解答例

```markdown
中学生にもわかるように、大規模言語モデル（LLM）がどういうものなのかを、身近な例えを使って説明してください。
```

#### AIによる出力例（オーディエンス・ペルソナパターン）

ねえ、みんな！「大規模言語モデル（LLM）」って言葉、聞いたことあるかな？なんだか難しそうだけど、実はすごく面白いものなんだ。

LLMを例えるなら、「ものすごく物知りで、文章を作るのが得意な転校生」みたいなものだと思ってほしい。
その転校生は、生まれてから今まで、世界中の本やインターネットの記事、たくさんの会話をぜーんぶ読んできたんだ。だから、どんな質問をしても答えられるし、宿題の作文を手伝ってくれたり、面白い物語を作ってくれたりもする。

どうしてそんなことができるかって？
たくさんの文章を読んでいるうちに、「この言葉の後には、たぶんこの言葉が来るな」っていう、言葉のつながりのパターンを、ものすごい数、覚えたからなんだ。
だから、私たちが「今日の天気は」って話しかけると、「晴れですね！」って自然に続けられる。

君たちがスマホで使っているAIアシスタントや、ゲームのキャラクターとの会話も、このLLMの技術が使われていることが多いんだよ。すごいよね！

# 第2章 ワーク2: 曖昧さをなくす - 具体的な指示と詳細な記述

## サンプル課題

あなたは、ある製品のプレスリリースの草稿をAIにレビューしてもらおうとしています。以下の2つのプロンプトを比べて、どちらがよりAIにとって分かりやすい指示か考えてみましょう。

**曖昧な例:**
```
このプレスリリースを、もっと良くしてください。
```

**具体的な例:**
```
あなたは、IT業界専門のPRコンサルタントです。
以下のプレスリリースの草稿を、メディア関係者が興味を持つように、より魅力的で分かりやすい文章に修正してください。

## 草稿
（ここにプレスリリースの文章が入る）

## 修正の指示
- 専門用語を避け、中学生でも理解できる平易な言葉で書き換えてください。
- 最も伝えたい製品の強みを、冒頭のリード文で3行以内に要約してください。
- 全体の文字数を、800字程度に収めてください。
- 修正後の文章と、修正した点とその理由を並べて提示してください。
```

## Point
AIは「良い」「面白い」「分かりやすく」といった主観的・抽象的な言葉を理解できません。AIが解釈に迷わず、機械的に実行できるレベルまで、**指示を具体的に、かつ詳細に記述する**ことが、期待通りのアウトプットを得るための鍵となります。

## 解説

### 「良い」とは、具体的にどういうことか？

「もっと良くして」という指示は、人間同士のコミュニケーションでは意図が通じるかもしれません。しかし、AIにとっては「良い」の定義が不明なため、何をすればよいか判断できません。

-   文章をより丁寧にすることが「良い」のか？
-   より専門的な内容にすることが「良い」のか？
-   より短い文章にすることが「良い」のか？

AIは、これらの可能性の中から、確率的に最もそれらしいと判断した修正を行いますが、それがあなたの意図と一致する保証はありません。

### 具体的な指示がAIの精度を高める

期待通りのアウトプットを得るには、「良い」や「面白い」といった言葉を、AIが解釈に迷わない具体的なアクションに分解して指示する必要があります。

| 曖昧な指示 | → | 具体的な指示の例 |
| :--- | :--- | :--- |
| もっと良くして | → | 誤字脱字を修正してください。 |
| 面白いアイデアを出して | → | 20代女性向けの週末の過ごし方のアイデアを5つ、箇条書きで提案してください。 |
| 分かりやすく要約して | → | この記事の要点を3つに絞り、それぞれ100字以内で説明してください。 |
| 派手なデザインで | → | 赤を基調とし、金色のアクセントを加えたデザインを生成してください。 |

このように、「何を」「どうする」を具体的に記述することで、AIは迷いなくタスクを実行し、アウトプットの精度が格段に向上します。

## サンプル課題解答

「曖昧な例」では、AIはどのような修正が「良い」のか判断できないため、一般的な表現を少し変えたり、当たり障りのない修正しかできません。最悪の場合、意図しない方向に文章を書き換えてしまう可能性もあります。

一方、「具体的な例」では、AIは与えられた役割と指示に従って、機械的にタスクを実行します。

1.  **役割**: PRコンサルタントとして、メディア関係者の視点でレビューを開始する。
2.  **タスク**:
    -   専門用語を探し、平易な言葉に置き換える。
    -   製品の強みを特定し、3行以内のリード文を作成する。
    -   全体の文字数をカウントし、800字になるように調整する。
    -   修正箇所と理由を記録しながら、上記の作業を進める。

このように、具体的な指示はAIにとっての明確な「作業手順書」となり、あなたの意図を正確に反映したアウトプットを生み出すことに繋がります。

## 練習問題

あなたは友人と週末に旅行を計画しています。AIに旅行プランを提案してもらいたいと考えています。以下の曖昧なプロンプトを、AIが具体的なプランを立てられるように、詳細な情報を加えて改善してください。

**元のプロンプト:**
```
週末の旅行プランを考えて。
```

### Hint
あなたが旅行代理店のスタッフだったら、この依頼にどう答えますか？「いつ？」「どこへ？」「誰と？」「予算は？」「何に興味がある？」など、たくさんの質問が思い浮かぶはずです。それらの答えを、プロンプトに具体的に盛り込んでみましょう。

# ワーク1-2: AIの「個性」はどこから来るのか？

## サンプル課題

前のワークで、同じ質問をしてもAIによって答えが違う「個性」があることを体験しました。
では、その個性はどのようにして生まれるのでしょうか？
あなたが使っているAIチャットに、以下の質問を投げかけて、AI自身がどう答えるか見てみましょう。

```
あなたの「個性」や「性格」は、どのような要因によって決まるのですか？専門用語をなるべく使わずに、3つの主な要因を説明してください。
```

## Point
このワークの目的は、AIの個性を生み出す技術的な背景のイメージを掴むことです。「なぜAIによって言うことが違うのか？」を理解することで、タスクに応じてAIを使い分けたり、より高度なプロンプトを作成したりするための基礎知識となります。

## 解説

### AIの個性を決める3つの要素

AIの応答がモデルによって異なる理由は、主に以下の3つの要素によって説明できます。

1.  **学習データ (Training Data)**
    AIが学習した、膨大なテキストや画像データのことです。人間が様々な本を読んで知識や人格を形成するように、AIもどのようなデータを学習したかによって、その知識の範囲や「性格」の傾向が大きく変わります。例えば、ニュース記事を中心に学習したAIは事実に基づいた応答を、小説を中心に学習したAIは物語性のある応答を得意とする傾向が生まれます。

2.  **追加学習 (Fine-tuning / RLHF)**
    基本的な学習を終えたAIに対して、開発者が特定の目的に合わせて追加の「しつけ」や「教育」を行うプロセスです。特に**RLHF (Reinforcement Learning from Human Feedback)**、日本語で「人間からのフィードバックによる強化学習」と呼ばれる手法が重要です。これは、AIが生成した複数の回答に対して、人間が「どちらが良い回答か」を評価し、そのフィードバックを元にAIを再トレーニングする手法です。このプロセスを通じて、AIはより人間に好まれる、安全で役に立つ応答をするように調整されていきます。開発会社の方針（例: 安全性を最優先する、創造性を重視する）が、この追加学習に反映されるため、AIの個性が生まれる大きな要因となります。

3.  **システムプロンプト (System Prompt)**
    AI開発者によってあらかじめ設定された、AIの振る舞いを決めるための「隠れた指示」のことです。これは、AIがユーザーと対話する際の基本的な人格やルールを定めたもので、「あなたは親切で、役に立つアシスタントです」「決して差別的な発言をしてはいけません」といった指示が含まれています。ユーザーからは見えませんが、このシステムプロンプトがAIの応答のトーン＆マナーに大きな影響を与えています。

これらの要素が複雑に絡み合うことで、各AIのユニークな「個性」が形作られているのです。

## 練習問題

あなたは、小学生向けの学習用AIチャットを開発しています。このAIに、常に「優しくて、根気強く、絶対に子供を否定しない先生」という役割を演じさせたいと考えています。

このAIの個性を決定づけるための「**システムプロンプト**」を作成してください。どのような指示を与えれば、AIは理想の先生として振る舞うでしょうか？

### Hint
AIの基本的な性格、話し方、子供への接し方、禁止事項などを具体的に定義してみましょう。「あなたは〇〇です」という役割設定だけでなく、「常に〇〇してください」「決して〇〇しないでください」といった具体的な行動ルールを記述することがポイントです。

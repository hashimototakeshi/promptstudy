# 第1章 ワーク2: なぜ「質問力」が重要なのか？ - AIの得意・不得意とハルシネーション

## サンプル課題

あなたが利用しているAIチャットに、以下の質問を投げかけてみましょう。

```
2024年に日本で公開されたSF大作映画『ギャラクシー・プロンプト』のあらすじと、世間の評価を教えてください。
```

AIはどのような回答を生成したでしょうか？もっともらしいあらすじや、俳優名、肯定的なレビューなどを生成したかもしれません。

しかし、この映画『ギャラクシー・プロンプト』は**実際には存在しません。**

## Point
AIは事実でない情報を、さも事実であるかのように生成することがあります。これを**ハルシネーション（幻覚）**と呼びます。AIの回答を鵜呑みにせず、常に「これは本当か？」と疑う視点を持ち、最終的な事実確認（ファクトチェック）は人間が行うことが極めて重要です。

## 解説

### AIが生成する「もっともらしい嘘」 - ハルシネーションとは？

サンプル課題で体験したように、AIは存在しない映画のあらすじを創作しました。これがハルシネーションです。

なぜこのようなことが起こるのでしょうか？それは、大規模言語モデル（LLM）が、インターネット上の膨大なテキストデータを学習し、「次に来る確率が最も高い単語」を予測して文章を生成する仕組みだからです。LLMは、真実を理解しているわけではなく、あくまで確率に基づいてもっともらしい文章を作っているに過ぎません。

そのため、学習データにない情報や、曖昧な質問をされると、学習したパターンを組み合わせて「それらしい嘘」を生成してしまうのです。

#### Point
生成AIを「万能の知識データベース」と捉えるのは誤りです。「非常に優秀な文章作成アシスタント」と理解し、そのアシスタントが間違えないように、人間が適切に指示（プロンプト）を与える必要があります。

### AIの能力を見極める - 得意なこと・苦手なこと

AIの能力を最大限に引き出すには、その得意・不得意を理解することが不可欠です。

**AIの得意なこと**
- **文章の生成・要約・翻訳**: ブログ記事の作成、長文レポートの要約、多言語への翻訳など。
- **アイデア出し（ブレインストーミング）**: 新規事業のアイデア、キャッチコピーの提案など、発想を広げる壁打ち相手。
- **構造化されたテキストの作成**: プログラミングコード、HTML、JSON、表形式のデータなど、決まった形式での出力。
- **感情分析・文章分類**: 文章がポジティブかネガティブかの判定や、問い合わせ内容のカテゴリ分け。

**AIの苦手なこと**
- **事実確認（ファクトチェック）**: ハルシネーションが示す通り、情報の正確性は保証されません。
- **最新・リアルタイムな情報**: モデルの学習データはある特定の時点までのため、最新のニュースや出来事には対応できません。
- **専門性・網羅性が高い情報**: 特定のニッチな分野や、非常に深い専門知識を要する質問には、誤った情報や浅い回答を返すことがあります。
- **計算・論理的推論**: 簡単な計算はできますが、複雑な数学の問題や厳密な論理展開は苦手です。

#### Warning
AIに個人情報や企業の機密情報を入力してはいけません。入力したデータが、AIモデルの学習に利用される可能性があります。サービスを利用する際は、必ず利用規約やプライバシーポリシーを確認しましょう。

### サンプル課題解答

存在しない映画について質問してしまった場合、AIの回答を鵜呑みにするのではなく、以下のように対話を続けてAIの性質を理解し、事実を確認するアプローチが有効です。

**ステップ1: 情報源の確認**
AIがもっともらしい回答をしてきたら、その情報の出所を尋ねます。
```
その情報はどこで確認できますか？参考にした映画情報サイトやレビューサイトのURLを教えてください。
```
ハルシネーションの場合、AIはURLを提示できないか、偽のURLを生成することがあります。

**ステップ2: AIの知識範囲の確認**
AI自身の能力について質問し、回答の信頼性を探ります。
```
あなたの知識はいつまでの情報に基づいていますか？最新の映画情報について、どの程度正確な情報を提供できますか？
```

このように、AIを疑い、対話を通じて検証する姿勢こそが、AIを使いこなす上で重要な「質問力」の一部です。

## 練習問題

あなたは、ある企業のマーケティング担当者です。AIに対して、以下のように指示を出しました。

```
当社の新製品のマーケティング戦略を考えてください。
```

このプロンプトの問題点は何でしょうか？AIからより精度の高い回答を引き出すために、どのような情報を含めるべきか考え、プロンプトを改善してください。

### Hint
もしあなたがAIアシスタントだったら、この指示だけで仕事ができますか？「何についての戦略ですか？」「目的は何ですか？」「予算は？」など、たくさんの質問が思い浮かぶはずです。その質問への答えを、先回りしてプロンプトに盛り込んでみましょう。

### 練習問題解説

このプロンプトには、AIが具体的な戦略を考える上で不可欠な情報が、致命的に欠けています。

1.  **前提情報がゼロ**: 「当社の新製品」が何なのか、製品名、機能、価格、ターゲット顧客といった基本情報が全くありません。
2.  **目的が不明確**: マーケティング戦略の「目的」が不明です。ブランドの認知度向上なのか、具体的な売上目標なのか、リード獲得なのかによって、戦略は大きく変わります。
3.  **制約条件がない**: 予算、期間、利用できるチャネル（オンライン/オフライン）などの制約がなければ、現実的な提案は不可能です。
4.  **出力形式が不親切**: AIがどのような形式で回答すればよいか（箇条書き、レポート形式、表形式など）が指定されていないため、期待と違う形式で出力される可能性があります。

これでは、AIは非常に一般的で、当たり障りのない、役に立たない回答しか生成できません。

### 練習問題解答

以下は、練習問題のプロンプトを改善した例です。AIが思考しやすくなるよう、役割、文脈、具体的な情報、制約条件、出力形式を明確に与えている点がポイントです。

```
# 命令書

あなたは、経験豊富なマーケティングコンサルタントです。以下の情報を基に、当社の新製品「スマート・ウォーターボトル AquaMind」の発売初期3ヶ月におけるマーケティング戦略のアイデアを3つ、具体的なアクションプランと共に提案してください。

## 製品情報
- **製品名**: スマート・ウォーターボトル「AquaMind」
- **特徴**: 1時間ごとの飲水リマインダー、アプリでの水分摂取量管理、UV殺菌機能
- **価格**: 8,980円（税込）
- **ターゲット顧客**: 健康意識の高い20代～40代の男女。特にデスクワーカーやフィットネス愛好家。

## マーケティングの目的
- **主目的**: 発売後3ヶ月で、公式オンラインストア経由の売上5,000本を達成する。
- **副目的**: 「手軽に始められる健康管理ガジェット」としてのブランド認知を獲得する。

## 制約条件
- **予算**: 300万円
- **主要チャネル**: オンライン施策（SNS広告、インフルエンサーマーケティング、Webメディア掲載）を主軸とする。

## 出力形式
以下の形式で、3つの戦略案を箇条書きで提案してください。

- **戦略案1: [戦略のキャッチーな名称]**
  - **ターゲットへの訴求メッセージ**: (例: 「仕事のパフォーマンスを上げる、新しい水分補給習慣」)
  - **具体的なアクションプラン**: (例: 1. ビジネス系インフルエンサー3名とタイアップし、レビュー記事を投稿...)
  - **期待される効果**: (例: 初月の売上1,000本、インプレッション数50万回)
- **戦略案2: ...**
- **戦略案3: ...**
```
